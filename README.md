# predict-model

# 全国新冠疫情预测模型

## 项目简介
基于机器学习的全国新冠疫情预测系统，使用历史数据预测未来疫情发展趋势。

## 功能特点
- 多省份数据整合分析
- 时间序列特征工程
- 多种机器学习模型
- 可视化预测结果

## 快速开始

### 环境要求
- Python 3.8+
- 依赖包：见requirements.txt

### 安装步骤
```bash
# 克隆项目
git clone https://github.com/你的用户名/covid-19-prediction.git

# 进入项目目录
cd covid-19-prediction

# 安装依赖
pip install -r requirements.txt
